# C++ STL
applying STLs with adding some custom methods in `custom-TL` namespace


##Dependencies:
1. cmake
2. make
3. g++

##To build:
1. go to `build` directory.
2. ```
   cmake ..
   ```
3. ```
   make
   ```
4.
   a. Ubuntu:
   ```
   ./STL_study
   ```
   b. Windows:
   ```
   .\STL_study/exe
   ```
